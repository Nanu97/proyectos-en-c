# proyectos-en-c
Resolucion de ejercicios 1er cuatrimestre

Este repositorio contiene distintos ejercicios y proyectos desarrollados en lenguaje C, creados como práctica durante el aprendizaje de programación estructurada.

La idea es funcionar como un gestor de ejercicios, donde cada archivo representa un proyecto o problema distinto, que puede ir creciendo y mejorándose con el tiempo.

# objetivo del repositorio
>>> Practicar lógica de programación
>>> Reforzar el uso de funciones
>>> Mejorar validaciones de entrada
>>> Mantener código claro y ordenado
>>> Servir como referencia de progreso personal

# en progreso
Este repositorio está en constante crecimiento. Se irán agregando nuevos ejercicios y mejoras a los existentes.
